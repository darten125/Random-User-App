# Random User App

Приложение для получения случайного пользователя. 
API: https://randomuser.me

---

## Функциональность

- Выбор пола и национальности для генерации пользователя
- Получение информации о случайном пользователе из сети
- Сохранение пользователей в локальную базу для просмотра списка
- Поддержка работы без интернета (кеширование)
- Экран детализации пользователя
- Навигация со стрелкой «Назад» на всех экранах кроме экрана с списком пользователей
- Обработка ошибок и отображение сообщения, если данные отсутствуют
- Прогресс-индикатор при загрузке данных

---

## UI

Интерфейс включает 3 основных экрана:

**Главный экран (генерация пользователя)**  
- Выбор пола (male/female)  
- Выбор национальности (например: US, DE, FR и др.)  
- Кнопка генерации нового пользователя  

**Экран данных пользователя**  
- Фото
- Личные данные
- Контакты (телефон, email)
- Адрес проживания

**Экран сгенерированных пользователей**  
- Отображение ранее сгенерированных пользователей
- Кнопка перехода на экран генерации пользователя
- Кнопка очистки всех сгенерировнных пользователей
- Открытие экрана данных конкретного пользователя при нажатии на одного из них
- Удаление конкретного пользователя при нажатии на кнопку удаления на карточке пользователя

## Архитектура

Использованы **Clean Architecture + MVVM**, со строгим разграничением слоёв:

- data — источники данных (API, Room), DTO + мапперы
- domain — бизнес-логика (UseCase, domain-модели, интерфейсы репозиториев)
- presentation — ViewModel, стейт менеджмент, UI

 ##  Технологии

- Язык разработки: Kotlin
- UI: Jetpack Compose
- Запросы к API: Retrofit + OkHttp
- Локальное хранение: Room
- DI: Koin
- Ассинхронность: Coroutines / Flow
